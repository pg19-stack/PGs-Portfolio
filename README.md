## Setup

No build tools required. This is a static site.

```bash
# Clone the repo
git clone https://github.com/pg19-stack/PGs-Portfolio.git

# Open locally
open index.html
```

## Adding Your Photo

Replace `photo.jpg` in the root directory with your professional headshot. The site includes a fallback "PG" monogram if no photo is present.
